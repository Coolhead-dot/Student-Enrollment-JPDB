# 🎓 Student Enrollment Form using JsonPowerDB

## 📌 Description

This project is a web-based Student Enrollment Form developed using HTML, JavaScript, and JsonPowerDB (JPDB). It allows users to store, retrieve, and update student data based on Roll Number (Primary Key).

---

## ⚙️ Features

* Add new student records (Save)
* Fetch existing records using Roll No
* Update student details
* Reset form to initial state
* Input validation for all fields

---

## 🗄️ Database Used

* JsonPowerDB (JPDB)
* Database Name: SCHOOL-DB
* Table Name: STUDENT-TABLE

---

## 🚀 Benefits of using JsonPowerDB

* Simple and fast NoSQL database
* Serverless architecture
* Easy API integration
* High performance
* No complex SQL queries required

## 📖 How to Use

1. Enter Roll No
2. If new → fill all fields → click Save
3. If exists → modify fields → click Update
4. Use Reset to clear form
